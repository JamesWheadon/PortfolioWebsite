Past Projects

Studying engineering at university gave me the oppurtunity to apply theory from lectures in practical projects. Below are a few of my favourites that I carried out at university, some with the resources available to apply the project to hardware and see them work in the real world. I still like to throw myself in to a project, some of my personal projects and projects done with futureproof are listed below.

Self-Balancing Footballer

A week-long project to make a controllable football player using lego mindstorms components, as well as MATLAB and Simulink to handle the control elements of the problem. The control system would stabilise the robot allowing it to balance on 2 wheels while being controlled to make it move forward and backwards, turn and kick a football.

Line Following Robot

Coding a line-following robot from scratch, given the documentation for the microcontroller of the robot and information about the other hardware like the camera, the robot was programmed to follow black lines and make decisions at junctions to navigate a simple environment.

Multi-Agent Path Finding AI

Implemented and developed multi-agent path finding artificial intelligences to plan paths for multiple robots in the same environment. The final algorithm found paths for non-holonomic heterogeneous robots with inertial constraints in continuous time, using a discrete representation of an environment that multiple robots are navigating simultaneously. The final algorithm was succesful in testing against leading methods and provably optimal in certain scenarios.
